# Proyecto API de Gestión de Citas Médicas

## Descripción
API para gestionar pacientes, doctores y citas médicas. Permite registrar pacientes y doctores, crear, cancelar y consultar citas, así como obtener estadísticas y disponibilidad de doctores.

---

## Requisitos
Para ejecutar el proyecto, necesitas tener instalado:

- **Node.js y npm** – Para ejecutar el servidor y manejar dependencias.  
- **Visual Studio Code** – Para escribir y depurar código.  
- **Git** – Para control de versiones y subir el proyecto a GitHub.  
- **Postman** (opcional) – Para probar las APIs.
